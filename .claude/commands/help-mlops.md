## Common Topics

### Tools
- **DVC**: Data Version Control - git for data/models
- **MLflow**: Experiment tracking, model registry, deployment
- **Docker**: Containerization for reproducible environments
- **FastAPI**: Modern Python web framework for APIs

### Concepts
- **Model drift**: When model performance degrades over time
- **Feature store**: Centralized repository for ML features
- **CI/CD for ML**: Automated testing and deployment pipelines
- **A/B testing**: Comparing model versions in production
